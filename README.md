# Dao De Jing Pattern Analysis Tool

An interactive web application for discovering geometric and structural patterns in the ancient Chinese text, the Dao De Jing.

## Features

- Character grid visualization (9,000+ characters across 81 chapters)
- Co-occurrence matrix analysis
- Radical highlighting system
- Context viewer for examining character usage
- Contrast pair analyzer for testing hypotheses
- Statistical validation tools

## Running Locally
```bash
npm install
npm start
```

## Data Source

Based on bamboo slip manuscripts and traditional texts.

## Research Methodology

This tool implements the Recursive Structural Model (RSM) framework for analyzing ancient texts as geometric notation systems rather than purely literary works.
